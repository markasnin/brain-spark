⚡ Brain Spark — Math Arena
A gamified math practice app for Israeli elementary school students (grades 1–6)
Built as a Progressive Web App with real-time Firebase backend, grade-adaptive questions, and a full achievement system.

📖 What Is This?
Brain Spark is a math game designed for Israeli kids in grades 1–6. Instead of boring worksheets, students earn points, build streaks, unlock topics, compete on a leaderboard with friends, and level up through a progression system — all while practicing curriculum-aligned math.
The game adapts entirely to the student's grade: question ranges, available categories, point values, and difficulty scaling are all grade-specific. A grade-1 student sees simple addition up to 30; a grade-6 student gets six-digit multiplication and complex fractions.
